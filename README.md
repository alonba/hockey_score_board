# 🏒 Rink Hockey Scoreboard

A web app for tracking and managing Rink Hockey matches. This app provides professional scoreboard functionality that can be installed on any device. All data is stored locally on your device. No information is sent to external servers.

## Download Instructions

This app can be installed as a Progressive Web App (PWA) on both iPhone and Android devices, giving you a native app-like experience without going through an app store.

1. Use your favorite browser and navigate to the scoreboard [website](https://alonba.github.io/hockey_score_board/).

2. On iPhones - Tap the **Share** button. On Android - Tap the **three-dot menu** (⋮) in the top right corner.

3. Tap **"Add to Home Screen"**. On Androids you might see **"Install app"**. Some Android devices show an install prompt automatically when you visit the site, look for a banner at the bottom saying **"Add Rink Hockey Scoreboard to Home screen"**.
